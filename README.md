# Enpour

> Come back for another round.

Enpour is a mobile-first platform that helps hospitality professionals maintain relationships with their guests through secure, permission-based connections.

## Development

```bash
npm install
npx expo start
```

## Web export

```bash
npx expo export --platform web
```

## Deploy

```bash
npx eas deploy --prod
```
